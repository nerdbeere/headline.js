Headline.js

by Julian Hollmann

Editable Headline

Code

myHeadline = $.headline($('.myHeadline'), {
    editable: true,
    lines: [
        { text: 'Welcome!', offset: 0 },
        { text: 'This is headline.js', offset: 30 }
    ]
});

Static Headline

Code

$.headline($('.myHeadline2'), {
    lines: myHeadline.getData()
});
